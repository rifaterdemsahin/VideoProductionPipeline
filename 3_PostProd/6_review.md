🎬 Video Review Strategy
- Conduct comprehensive quality checks
- Gather feedback from stakeholders
- Ensure technical standards are met

🕰️ Previous Retro
- Loom effective for remote reviews
- Audio checks critical at multiple stages
- Multiple review passes catch more issues

🔍 Required Inputs
- 📹 Raw footage
- 🎵 Music/audio tracks
- ✍️ Edit notes

🎯 Review Checklist

⏱️ Initial Review
1. 🖥️ Technical Check
   - Review in Canva
   - Verify file formats
   - Check copyright status
   - Document initial findings

2. 📺 Content Review
   - Watch on big screen
   - Test in Zoom meeting
   - Share Loom recording
   - Gather feedback

3. 🎬 DaVinci Review
   - Normalize audio levels
   - Check transitions
   - Fix framing issues
   - Review mini cuts

4. 🔊 Audio Review
   - Test at 30db volume
   - Check from 3m distance
   - Verify music placement
   - Document audio fixes

5. 📋 Final Steps
   - Create review copies
   - Upload to Loom
   - Share with team
   - Bookmark feedback
   - Update file names
